Pivot Scheduler

Send Pivot view (Business Intelligence Report) automatically through pre-defined users by Minutes, Hours, Days, Months.

Tested on Community & Enterprise

Features:

  • Reports automatically will send to the scheduled users by Minutes, Hours, Days, Weeks, Months.
  • Reduce the manual work like download, compose and attach to send.
  • We can add unlimited scheduled reports.
Add your desired Filters, Group By, Time Ranges to Favorites from any Pivot view.
Activate the developer mode
  • Go to Settings.
Create Scheduler
  • Go to Settings > Automation > Scheduled Pivot Reports.
  • Create / Edit a record.
  • Fields:
  • Model: Can select the models which has pivot view.
  • User-defined Filter: Can select user-defined filter/favorite from selected model's pivot view.
  • Recipients: Auto filled and able to select the users who has access to the selected model.
  • Related Cron: The cron is create/delete/update when the this configuration record is create/delete/update.
  • Action(s):
  • Button in header "Send Manually": Click thi to send manually.
